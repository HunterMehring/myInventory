Instructions

Inventory

This is where you can see all of the items that you have created. To create an item you first need to create a category. You can create a category once you are on the inventory page. Think of Categories as “types of items”. If you wanted to add an XL jacket number 13 you would first need to create the category “jacket”. Once you create a category you will be able to select it from a dropdown when you are creating items.

On the items page you can create, remove, and edit your items. Items are sorted by category and then size. Items are separated by whether they are in storage or out of storage. Once you give an item to a player it will go into out of storage. Items can only be removed if they are in storage.

Players

This is where you can give items to players. Click on a player to see what items they have. The add/remove items button takes you to where you can see all items available to take. You also have the option for players to give items back.

Search

If you need to find an item, simply enter in the category and number and a highly sophisticated algorithm gives you all the useful information you need to know.

Tasks

This allows the user to assign tasks to players. This is intended to help coaches remember what each player needs to get done. Click on the task name to see who has or has not completed each task.

Changes

This is where you can see every change that has taken place in the app. Every time you add a player, remove a category, give a task to a player, or anything else within the app, we make a record of it. You can click each of the categories to see which type of change you would like to see.
